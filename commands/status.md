# Project Status Report

Execute comprehensive status check using multiple agents:

1. **Task Status** (task-manager)
   - Active tasks and their DDRIVE phases
   - Task completion rate
   - Blocked tasks

2. **Sprint Progress** (sprint-planner)
   - Current sprint metrics
   - Velocity and burndown
   - Team capacity

3. **Milestone Status** (milestone-tracker)
   - Upcoming milestones
   - Risk assessment
   - Critical path

4. **Progress Metrics** (progress-monitor)
   - Real-time progress
   - DDRIVE compliance
   - Phase distribution

5. **Quality Metrics** (coverage-analyzer)
   - Test coverage
   - Code quality score
   - Technical debt

Generate unified dashboard with all key metrics and recommendations.
